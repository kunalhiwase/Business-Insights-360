# Business Insights 360

A comprehensive **Business Intelligence solution** built for **AtliQ Hardware** to replace Excel-based reporting with an interactive **Power BI** dashboard. This project leverages **MySQL, SQL, Power Query, and DAX** to deliver department-specific insights across Finance, Sales, Marketing, Supply Chain, and Executive reporting. It also includes complete **Business Analyst documentation** including Project Overview, BRD, FRD, and Problem Statement.

# Key Features

- Power Query ETL Pipeline
- Dynamic DAX Measures & KPIs
- Department-wise Dashboard Views
- Benchmark Comparison (Target vs Last Year)
- Customer & Product Performance Analysis
- Forecast Accuracy & Risk Analysis
- Interactive Drill-through & Tooltips
- Power BI Service Deployment
- Auto Refresh via Gateway
